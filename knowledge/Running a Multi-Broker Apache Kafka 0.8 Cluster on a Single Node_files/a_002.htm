                                                                                                Kostya Golikov Jeff Gregory 9 years ago edited

                                                                                                Most likely you're running on Linux box with encrypted filesystem. Scala compiler generates a lot of files with SomeClass$SomeMethod lengthy names to map it's closures and other such stuff to JVM classfiles (and Java too, but it does it much less frequently -- for anonymous classes), so it is easy to run out of such filesystems filename length limit. The most easier you can do is to try to run build in un-encrypted directory. There is also -Xmax-classfile-name scala compiler option.
